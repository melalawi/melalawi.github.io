Global Twitch Emotes

Summary

Global Twitch Emotes was my first browser extension, and subsequently my first foray into the realm of web programming. It came from my desire tois a simple extension that finds and replaces all Twitch.tv emote phrases with their appropriate icons. It supports Kappa, global, subscriber, BetterTTV, and now FrankerFaceZ emotes everywhere!

Global Twitch Emotes This was the first bit of web code that I'd written. Twitch, the immensely popular game streaming website, has a wide variety of emoticons that are quickly becoming wide-known in the gaming community. This extension allows for their use anywhere on the web. It has over 4000 users across Chrome and Firefox and is particularly popular on Reddit. Though this extension was made purely for entertainment, I learned much while creating it.
